!!!
The `os` module is just a bunch of JSONRPC wrappers.
Their implementations depend on the host language.
!!!

### `os.listdir(path)`

List all files and directories in a directory.

### `os.mkdir(path)`

Create a directory.

### `os.rmdir(path)`

Remove a directory.

### `os.remove(path)`

Remove a file.

### `os.path.exists(path)`

Check if a file or directory exists.
